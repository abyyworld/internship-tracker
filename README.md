## ⚙️ How this works
- A Python scraper (`internship_watcher.py`) runs **every morning via GitHub Actions** — no server, no computer needed.
- It pulls [SimplifyJobs](https://github.com/SimplifyJobs), [vanshb03](https://github.com/vanshb03/Summer2027-Internships), sndsh404, and 21 companies' Greenhouse job APIs.
- Full data (with your own status/notes columns) lives in [tracker.csv](tracker.csv). Manual-check links in [manual_checks.md](manual_checks.md).
- Want your own copy? Fork this repo and enable Actions. Zero dependencies — Python stdlib only.
